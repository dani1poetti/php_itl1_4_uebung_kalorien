## Kalorienrechner Überblick

Mein Programm / Anforderungen:
> - Schönes Frontend<br>
> - Benutzer kann seine Werte eingeben<br>
> - Benutzer kann keine ungültigen Werte eingeben<br>
> - Nach Absenden des Formulares, wird im Hintergrund der Grundumsatz
berechnet und zusätzlich der Umsatz durch Bewegung etc.
> - Der Gesamte Umsatz wird dann wieder schön ausgegeben
> - Richtigkeit der Berechnung wurde überprüft
